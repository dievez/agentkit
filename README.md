# AgentKit

Open-source Next.js starter for shipping AI chat apps fast.

AgentKit gives you the free foundation: auth, streaming chat, multi-model support, and a clean dashboard scaffold. When you need the production revenue stack — RAG, LangGraph agents, billing, multi-tenancy, observability, and support operations — upgrade to **AgentKit Pro**.

## Why this repo exists

Most AI starter kits force a bad tradeoff:
- good AI features, but no SaaS infrastructure
- good SaaS infrastructure, but no serious AI stack

**AgentKit** is the free OSS entry point.
**AgentKit Pro** is the paid version for founders who want both.

## What you get in the free tier

- Next.js App Router scaffold
- Supabase auth integration
- Streaming AI chat
- Multi-model support
- Clean dashboard + chat UX
- MIT-licensed public codebase

## Upgrade path: AgentKit Pro

AgentKit Pro adds the production layer that serious AI SaaS products need:

- RAG pipeline with pgvector
- LangGraph agents + tool calling
- Credits-based billing
- Multi-tenancy + org isolation
- LLM observability dashboard
- Ollama / local model support
- Built-in support ticket workflows
- Lifetime updates via private GitHub access
